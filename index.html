Airtame Setup Instructions
  1. Log into Airtame Cloud at airtame.com
  2. Select the Bullpen device → Digital Signage → Add Website App
  3. Paste the URL for each dashboard page (e.g., https://your-username.github.io/ridgehouse-dashboards/scorecard-dashboard.html)
  4. Set display time per slide (15-20 seconds recommended)
  5. Repeat for the Conference Room device with its playlist
  6. Set display schedule: 7:00 AM – 7:00 PM, weekdays
